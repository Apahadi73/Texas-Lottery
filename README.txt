# Texas-Lottery
A native android app that prints random lottery numbers for the user to play

# Credit/Information:
* The idea for this app was taken from Lottery Number Generator (link: https://www.calculatorsoup.com/calculators/games/lottery-number-generator.php)
* This app mimics the Texas Lottery's quick pick functionality while giving users option to select any Texas Lottery games along with number of plays

# How to Play?
* Home Activity - user can select the game from the game dialogue and number of plays. The user then should press "Generate" button when they are done.
* Result Activity - The results are displayed in list form. The user can play the new game if they press "New Game" button
